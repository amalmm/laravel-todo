Laravel todo example

## Installation
change
```shell
.env.exmple to .env
```
create demo-app, in db mysql
```php
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=demo-app    
DB_USERNAME=root
DB_PASSWORD=

```

## run

```shell
php artisan serve
```
```shell
php artisan migrte
``` 
 
 
## url
index 
```shell
http://127.0.0.1:8000/todo
```
